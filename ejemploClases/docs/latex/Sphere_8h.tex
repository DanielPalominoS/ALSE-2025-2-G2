\doxysubsection{Detailed Description}
Declaration of the \mbox{\hyperlink{classSphere}{Sphere}} concrete 3D shape. 

The \mbox{\hyperlink{classSphere}{Sphere}} class implements the \mbox{\hyperlink{classShape3D}{Shape3D}} interface and provides methods to compute the volume and surface area of a sphere. The class stores the sphere size in the private member {\ttfamily radious} (the radius of the sphere).
