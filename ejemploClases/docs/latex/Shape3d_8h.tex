\doxysubsection{Detailed Description}
Abstract base class for 3D shapes. 

This header defines the \mbox{\hyperlink{classShape3D}{Shape3D}} interface which provides two pure virtual methods that all concrete 3D shape classes must implement\+: volume and surface area calculations.
