\doxysubsection{Detailed Description}
Concrete implementation of a three-\/dimensional sphere. 

\mbox{\hyperlink{classSphere}{Sphere}} represents a geometric sphere defined by its radius. It implements the \mbox{\hyperlink{classShape3D}{Shape3D}} interface by providing concrete implementations for computing volume and surface area.
