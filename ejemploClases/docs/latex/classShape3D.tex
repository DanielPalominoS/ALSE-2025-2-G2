\doxysubsection{Detailed Description}
Interface for three-\/dimensional geometric shapes. 

\mbox{\hyperlink{classShape3D}{Shape3D}} is an abstract base class that requires derived classes to implement methods for computing the volume and the surface area of a shape. It defines a virtual destructor to ensure proper cleanup of derived objects through base pointers.
